# vue-template-cardboard
Turn Colorlib cardboard template into vue UI application for future use in client side.  As soon as server is built this app will be served and updated according to server needs.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
